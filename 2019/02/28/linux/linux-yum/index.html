发表于 2019-02-28

前言

在没有激活的机器上使用yum命令无法使用，临时替换免费版。推荐使用正版！

步骤

  1. 检查已经安装yum包

  2. 删除已安装的yum包

  3. 下载yum相关包
    浏览器打开阿里镜像站,网址可能会变

  4. 安装yum

  5. 下载源地址文件

  6. 备份原源地址文件

  7. 修改阿里源地址文件

    7.6.1810为版本号

  8. 清理缓存和重新生成缓存
