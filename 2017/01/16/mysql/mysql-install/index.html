记录Windows 系统安装mysql数据库

发表于 2017-01-16 更新于 2017-01-17 分类于 Mysql

环境

  • Windows10 操作系统
  • mysql 安装包

下载mysql安装包

在官网下载社区版。本文安装的是mysql5.6。

安装

直接双击运行即可

根据情况选择安装类型


这个地方需要注意的是，如果要自定义安装和数据路径，一定要选择自定义选项。

根据自己的情况选择选择产品和功能

选择自定义的安装和数据路径


检查要求时直接点击执行按钮即，执行完成点击下一步

接下来就是真正的安装了，点击执行

点击下一步进行配数据库配置

  1. 配置服务器类型和网络
  2. 配置账户和角色
  3. 配置访问数据库权限
  4. 配置Windows服务
点击执行应用服务器配置

测试是否安装成功
  1. 打开命令窗口，输入密码
  2. 选择test数据库，操作正常，说明我们安装成功了。

结语

本文记录了最简单的安装过程，适用于学习mysql初期使用。
